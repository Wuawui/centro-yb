# Ideas — CentroYB
## Features futuras y mejoras del producto

---

## Corto plazo (post-deployment)
- [ ] Dashboard web para admin (ver pacientes, citas, cobros desde browser)
- [ ] Onboarding automático (formulario web → config.json generado)
- [ ] Multi-idioma (español/inglés/portugués)
- [ ] Plantillas de mensajes customizables por centro
- [ ] Exportar reportes a PDF/Excel

## Medio plazo
- [ ] Multi-centro (una instancia, múltiples sucursales)
- [ ] App móvil para terapeutas (ver agenda del día)
- [ ] Pagos online integrados (link de pago en WhatsApp)
- [ ] Encuestas de satisfacción post-sesión automáticas
- [ ] Integración con Google Calendar
- [ ] Historial clínico por paciente (notas del terapeuta)

## Largo plazo
- [ ] IA para responder preguntas clínicas frecuentes
- [ ] Telemedicina (videollamada desde WhatsApp)
- [ ] Marketplace de terapeutas (conectar centros con profesionales)
- [ ] API REST para integraciones externas (contabilidad, CRM)
- [ ] White-label (otras agencias lo revenden con su marca)
- [ ] Franquicia del modelo a otros países LatAm

## Ideas de monetización adicionales
- Setup fee por centro nuevo ($50-100)
- SMS/WhatsApp bulk (costo por mensaje extra)
- Templates premium de mensajes ($5/mes add-on)
- Integración contable (exportar para contador)
- Certificado SSL incluido en plan premium
